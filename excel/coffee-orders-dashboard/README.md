# Coffee Orders Sales Dashboard

## Overview

This project is an interactive sales dashboard built in Microsoft Excel that analyzes coffee order data to provide insights into sales performance, customer behavior, and product trends.

The dashboard enables users to explore the data using interactive filters and visualizations, making it easier to identify key business metrics and patterns.

> **Note:** This project was completed by following the tutorial by Mo Chen on YouTube as part of my learning journey in Excel data analysis and dashboard development. While guided, I built the dashboard myself to practice Excel techniques and gain hands-on experience.

## Objectives

- Clean and prepare sales data
- Analyze customer orders
- Create interactive dashboards
- Visualize sales trends
- Practice Excel data analysis techniques

## Skills Demonstrated

- Data Cleaning
- Excel Formulas
- XLOOKUP
- INDEX & MATCH
- PivotTables
- Pivot Charts
- Slicers
- Timelines
- Conditional Formatting
- Interactive Dashboard Design

## Dashboard Features

- Sales by Country
- Sales by Coffee Type
- Monthly Sales Trends
- Top Customers
- Interactive Slicers
- Timeline Filter
