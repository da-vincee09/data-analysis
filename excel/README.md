# Microsoft Excel Projects

This directory contains my Microsoft Excel data analysis portfolio, showcasing projects focused on transforming raw data into meaningful business insights through analysis, visualization, and interactive dashboards.

## Overview

The projects in this folder demonstrate practical Excel skills commonly used in business analytics and reporting. They cover the complete data analysis workflow—from data preparation to presenting actionable insights through professional dashboards.

## Skills Demonstrated

- Data Cleaning and Preparation
- Data Analysis
- Exploratory Data Analysis (EDA)
- Dashboard Development
- Data Visualization
- KPI Reporting
- Business Reporting
- Data Transformation
- Trend Analysis

## Excel Features Used

- Formulas and Functions
- PivotTables and Pivot Charts
- Slicers and Timelines
- Conditional Formatting
- Named Ranges
- Dynamic Charts
- Power Query
- Data Validation
- Tables
- Interactive Dashboards

## Folder Structure

Each project in this directory will typically include:
